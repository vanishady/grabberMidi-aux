### Workaround for Focusrite Scarlett 6i6
1. install driver (Focusrite Control)
2. in Windows, select input device: **Microphone (Scarlett 6i6)**
3. in Windows, select output device: <**Your PC Speakers**> (e.g. BenQ, Realtek ...)
4. Run `python recorder.py`
5. Audio Input Device: *Analogue 1+2 Focusrite USB Audio*
